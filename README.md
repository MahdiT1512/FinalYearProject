# CodeSpark

CodeSpark is a mobile code-learning app that uses gamification to teach programming concepts effecitvely. It is designed to engage users and help them learn Python through bite-sized lessons, burst syntax exercises and its unique Lesson Deck feature.

##Features of CodeSpark

- The learn tab presents content through lessons broken down into a slide or card format
- The syntax tab provides exercises and activities to help with keyword and syntax mastery
- The projects tab provides project ideas for programmers looking to apply skills learnt through the app practically
- Badges and avatar gamification systems present along with a deep reward loop
- The ranks tab uses a leaderboard allowing other users to compete against each other, in weekly, monthly and all-time XP

## Prerequisites

- Node.js
- npm
- Optionally if wanting to use CodeSpark on a mobile phone, Expo Go app can be used


## Installation

Clone the repository:
git clone https://github.com/MahdiT1512/FinalYearProject.git

cd BasicImplementation/codeApp

Then with npm:
npm install

Then run the command:
npx expo start --clear

To access CodeSpark as a website you can simply go to (UI may break or not be as smooth as it is not designed as a website)
http://localhost:8081/

To access CodeSpark on a mobile device download the Expo Go app:
Switch from development build to Expo Go in terminal by pressing s 

Use the Expo Go app to scan the given QR code.

Once page has loaded press on the familiar directory ending in /codeApp that appears(may be confusing due to phone formatting, but the button with the directory is clickable)
This opens CodeSpark on the phone.
